## Глоссарий терминов (без названий источников — правило 0 #292)

| Термин | Объяснение (для подписчика, не для владельца) |
|---|---|
| **OB (Order Block)** | Зона на графике, где крупный игрок, вероятно, набирал/сбрасывал позицию — цена часто возвращается туда перед продолжением движения |
| **FVG (Fair Value Gap)** | Разрыв цены между свечами — рынок часто «дозаполняет» такие разрывы прежде чем двигаться дальше |
| **BOS / CHoCH** | Пробой структуры рынка (BOS — по тренду, CHoCH — смена тренда) — сигнал, что направление подтвердилось или начало разворачиваться |
| **Свип ликвидности** | Резкий прокол уровня (стоп-охота) с быстрым возвратом — часто предшествует настоящему движению в обратную сторону |
| **Killzone** | Окно повышенной активности рынка (открытие Лондона/Нью-Йорка и т.п.) — не все часы суток одинаково значимы |
| **DCA-вход (лесенка)** | Вход не одной суммой, а частями по нескольким уровням цены — снижает риск неудачного тайминга |
| **R:R (риск/прибыль)** | Соотношение потенциальной прибыли к риску на сделку — чем выше, тем меньше нужно угадывать направлений, чтобы быть в плюсе на дистанции |
| **Чеклист N/6** | Сколько из 6 фильтров методологии подтвердились для конкретного сигнала — честная мера силы сетапа, не гарантия исхода |
| **PD (фаза цены)** | Положение цены относительно суточного якоря (например, выше/ниже полуночи по Нью-Йорку) — часть определения текущей фазы рынка |
