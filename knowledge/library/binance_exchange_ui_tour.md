# Binance 1-3 — платформенный тур биржи (та же категория, что уже конспектировано)

**Три коротких файла (286+205+262 сек), чистый UI-туториал Binance**:
фьючерсный терминал (изолированное/кросс плечо, лимит/маркет ордера, TP/SL,
перевод между кошельками спот→фьючерсы), обзор типов кошельков (fiat/spot/
margin/futures/P2P), главная страница биржи (обменник, покупка карты/P2P,
API-ключи для сторонних платформ вроде ATAS). Полностью совпадает по жанру с
уже законспектированным `exchange_onboarding_order_types_leverage.md` (батч
10) — платформенная механика, не торговая методология.

**Единственная конкретная цифра**: рекомендация автора — изолированное плечо
15x фиксированно, риск на сделку 1% (согласуется с уже известным
риск-менеджментом, не новое).

**Модуль бота**: нет — чистый платформенный онбординг для человека-подписчика,
не относится к анализу/сигналам бота.
