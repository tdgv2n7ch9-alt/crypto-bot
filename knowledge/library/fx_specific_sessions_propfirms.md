# 28. Стрим Индикаторы и особенности торговли FX (поток 7.0, лектор "Гарри"/Инструктор D)

**Честно вне скоупа бота почти целиком.** Урок explicitly про ФОРЕКС-специфику
(валютные пары, торговые сессии с открытием/закрытием по часам, проп-фирмы/
challenges, экономический календарь новостей) — инструктор САМ явно
противопоставляет это крипторынку: "очень важно это отличие от крипторынка,
потому что крипту мы торгуем 24 на 7... с Форексом так не пойдет, на Форекс
есть только 2 зоны повышенной волатильности (9-12, 14-17 GMT+2)". То есть
структура сессий с открытием/закрытием НЕ переносится на крипту по прямому
признанию источника — не gap, осознанное несоответствие доменов.

**Killzone-часы (9-12 при-Лондон/Лондон, 14-17 при-Нью-Йорк/Нью-Йорк, GMT+2,
lunchtime 12-14 = мёртвая зона)** — специфичны для форекса (открытие
региональных бирж/банков), а не универсальный крипто-концепт — форекс-версия
уже известных killzone-часов, не создаёт нового вывода для крипто-бота (крипто
killzone-часы уже отдельно задокументированы, см. `sancho_8_lesson_course.md`).

**Прочее содержание (проп-фирмы/challenges, комиссии, вывод средств)** —
организационные/финансовые детали работы с форекс-брокерами и
проп-трейдинг-фирмами, не относится к торговой методологии вообще.

**Модуль бота**: нет — форекс-специфичный контент, крипто-бот форекс не
торгует.
