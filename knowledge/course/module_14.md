# Модуль 14. VWAP

## Урок 59. Расчёт и смысл

**VWAP** (Volume Weighted Average Price) — средняя цена, взвешенная по объёму:

VWAP = Σ(типичная цена × объём) / Σ(объём), где типичная цена = (High + Low + Close) / 3

Показывает «справедливую» среднюю цену сессии — ориентир для оценки своего входа.

## Урок 60. Сигналы и ограничения

- Цена **выше VWAP** — покупатели контролируют сессию (бычий контекст), ниже — продавцы
- Крупные игроки исполняют большие ордера, ориентируясь на VWAP: купить ниже VWAP = купить лучше рынка
- Ограничения: индикатор **запаздывает** и работает **внутри одной сессии** — между сессиями теряет точность
