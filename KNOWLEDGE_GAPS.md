# KNOWLEDGE_GAPS.md — честный список того, что НЕ подключено бесплатно

Заведён в Пакете 3 «НОЧНОЙ», М2 (2026-07-11), по прямому требованию владельца:
"каждый эндпоинт проверить живьём ДО написания кода", "недоступное бесплатно —
честно занести сюда". Формат: находка + источник (прямая ссылка) + дата
проверки + вердикт. Правила Протокола правды (`CLAUDE.md`) действуют
полностью: без ссылки факт не заносится.

## On-Chain метрики (Фаза C)

### Подключено бесплатно (Пакет 3 М2, `onchain_metrics.py`)

Каждый пункт проверен живым запросом (curl) 2026-07-11 непосредственно перед
написанием кода — сырые ответы см. в `PROGRESS.md`, чекпоинт Пакета 3 М2.

- **mempool.space** — комиссии сети BTC (`/api/v1/fees/recommended`),
  congestion мемпула (`/api/mempool`), прогресс до пересчёта сложности
  (`/api/v1/difficulty-adjustment`). Без ключа, без лимита в документации на
  момент проверки. [проверено — 2026-07-11, живой curl, HTTP 200]
- **blockchain.info charts API** — хешрейт, сложность, доход майнеров BTC
  (`api.blockchain.info/charts/{hash-rate,difficulty,miners-revenue}`). Без
  ключа. [проверено — 2026-07-11, живой curl, HTTP 200]
- **DeFiLlama** — глобальный TVL всех сетей (`api.llama.fi/v2/historicalChainTvl`),
  суммарный circulating supply стейблкоинов
  (`stablecoins.llama.fi/stablecoincharts/all`). Без ключа. [проверено —
  2026-07-11, живой curl, HTTP 200]
- **alternative.me Fear & Greed Index** (`api.alternative.me/fng/?limit=1`) —
  уже используется в нескольких карточках `bot.py` (Обзор, Институционал) с
  более ранних этапов; в Пакете 3 М2 добавлен отдельный самостоятельный фетч
  для карточки On-Chain (`onchain_metrics.fetch_fear_greed()`). Без ключа.
  [проверено — 2026-07-11, живой curl, HTTP 200]

### НЕ подключено — платно или не подтверждено бесплатным

- **SOPR / MVRV / NVT / Puell Multiple / LTH-STH supply (Glassnode)** —
  Glassnode НЕ имеет бесплатного API-тира. `docs.glassnode.com/basic-api/api`:
  программный доступ начинается с Advanced ($49/мес), даёт только урезанный
  "Light API" (14 дней истории, 1d-разрешение, 50 запросов/день).
  `studio.glassnode.com/pricing`: та же страница утверждает, что API вообще
  не включён в Advanced — только в Professional (custom pricing). Источники
  расходятся в деталях, но сходятся в главном: тира с $0 и программным
  доступом нет. [проверено — 2026-07-11 (перепроверка находки Пакета 2 М5),
  WebFetch на обе страницы] — **вердикт: платно, решение по бюджету за
  владельцем**.
- **SOPR / MVRV / NVT (BGeometrics / bitcoin-data.com)** — подтверждён
  $0/мес тир (10 запросов/час, 15/день, история 4 года), сайт заявляет
  SOPR/MVRV/NVT где-то в API. НЕ подтверждено, какие метрики входят именно
  в бесплатный тир против платных (Advanced+) — страница пейволла явно
  показывает часть данных только с Advanced, про распределение
  SOPR/MVRV/NVT/Puell/LTH-STH по тирам умалчивает. [не удалось проверить до
  конца — Пакет 2 М5, не переисследовалось в Пакете 3 М2, т.к. Пакет 3 явно
  ограничил скоуп источниками blockchain.com/mempool.space/DeFiLlama/
  alternative.me] — **вердикт: требует отдельной живой проверки конкретных
  метрик в бесплатном тире, если владелец решит идти этим путём вместо
  Glassnode**.
- **Puell Multiple, LTH/STH supply — ЛЮБОЙ источник** — на момент проверки
  (Пакет 2 М5, WebFetch) не найдено упоминания на главной странице API
  BGeometrics; отдельно не искалось на других бесплатных сервисах в Пакете 3
  (вне скоупа задания). [данных недостаточно] — **вердикт: источник не
  найден, требует отдельной разведки, если останется приоритетом**.

## ETH-специфичные ончейн-метрики

- **Etherscan (whale-трекинг ETH)** — владелец обозначил его как отдельный,
  более поздний пакет ещё в ответах к Пакету 2 ("В этот пакет входит ТОЛЬКО
  каркас+Glassnode; Etherscan и DeFiLlama — следующие пакеты"). Задание
  Пакета 3 М2 не включает Etherscan в список источников — не рассматривался.
  **Статус: НЕ начат, ждёт отдельного пакета**.
- В Пакете 3 М2 для символа `ETH` (и любого, кроме `BTC`) `onchain_metrics.
  get_free_onchain_snapshot()` честно возвращает `btc_chain: None` —
  hashrate/difficulty/mempool-метрики привязаны к сети BTC, для других
  символов бесплатного ончейн-источника в этом пакете нет. Рыночные метрики
  (F&G, DeFiLlama TVL/стейблкоины) не привязаны к символу и показываются
  всегда.

## Формула shadow-скоринга по этим метрикам

Ещё НЕ спроектирована — `onchain_metrics.shadow_score_adjustment_free()`
честно возвращает `available: False` вне зависимости от того, получены ли
данные. Дизайн формулы (какой вес дают hashrate/TVL/F&G сдвигу shadow-score)
— отдельный шаг, требующий явного решения владельца (боевой скоринг
затрагивать нельзя без одобрения — граница `CLAUDE.md`).
