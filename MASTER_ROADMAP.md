## Правило эталона

Непоколебимый образец: состав меняется только новой версией по решению
владельца. Цвета — только по факту DoD («в бою» = протестировано +
задеплоено + живая проверка). Исключённое в знаменатель не входит. Защита —
отдельный контур, не раздувает шкалу сигналов. Единственный выход системы —
ТОЧКА (символ, сторона, спот/фьючерс, DCA-зона, SL, TP, R:R≥1.5, риск
1–3%); метрика каждого пакета — стал ли сигнал точнее / правдивее / раньше.
Протокол правды — к каждой строке.
